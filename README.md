# UploadFileRPA
This project tries to find various solutions on how to upload files using automation tools.

flaskFormWithUploadFile
--
In this project you will have a folder with a flask microproject, once loaded you will have a form with two text fields and an input control to upload files.

UploadFilesHeadlessRPA
--
In this folder you will have a bot, RPA Python BotCity and another example script in Selenium. Both scripts use the chrome driver.
